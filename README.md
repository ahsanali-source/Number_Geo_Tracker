# Number_Geo_Tracker
PhoneLocator is a simple Python project that allows users to find the location and carrier of any phone number. The project uses phonenumbers for carrier and geolocation lookup, OpenCage API for precise coordinates, and Folium to display the location on a map.

Features:
Get the country or location of a phone number
Find the carrier/service provider
Convert location into latitude and longitude using OpenCage
Display the numberâ€™s location on an interactive map
Save the map as mylocation.html

Requirements:
1. Python
2. Libraries: phonenumbers, opencage, folium
3. OpenCage API key (replace 'your_opencage_api_key' in code)

How to Run:
1. Clone or download the project

2. Install libraries:
  pip install phonenumbers opencage folium

3. Add your OpenCage API key
4. Run the Python file
5. Enter the phone number in myphone.py
6. Open mylocation.html to view the map

Editor:
Developed and tested using Visual Studio Code (VS Code). Open the project folder in VS Code to run or edit the code.
